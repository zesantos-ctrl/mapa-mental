🧠 Mapa Mental: HTML e CSS

HTML & CSS

HTML

  • Estrutura básica:

    Todo HTML começa com a estrutura abaixo:

    <!DOCTYPE html>
<html>
  <head>
    <title>Título</title>
  </head>
  <body>
    Conteúdo
  </body>
</html>
  • Tags Comuns:

    Usamos tags como <p> para parágrafos e <a> para links.

    <p>Olá mundo!</p>
<a href="https://google.com">Google</a>
  • Semântica:

    Tags como <header>, <main> e <footer> ajudam a organizar melhor o conteúdo.

  • Atributos:

    Atributos adicionam detalhes extras.

    <img src="imagem.png" alt="descrição">

CSS

  • Estilização básica:

    CSS controla cor, tamanho, fonte e layout.

    body {
  background-color: #f0f0f0;
  color: #333;
}
  • Box Model:

    Define espaço ao redor dos elementos.

    div {
  margin: 10px;
  padding: 20px;
  border: 1px solid #000;
}
  • Layout com Flexbox:

    Distribui itens em linha ou coluna.

    .container {
  display: flex;
  justify-content: space-between;
}
  • Responsividade:

    Deixa o site adaptável para celular, tablet, etc.

    @media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}
